# tht-gh-search

search github to find public repositories from the term

## Run
### Server
```bash
$ bundle install
$ bin/rails server
```

### Tests
```bash
$ bin/rails spec

# Or
$ bundle exec rspec
```

### Rubocop
```bash
$ bin/rails cop

# Or
$ bundle exec rubocop
```
